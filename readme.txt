A plugin that adds a Persian date picker field to Advanced Custom Fields.

== Description ==

ACF Persian Dropdown Calendar adds a dropdown-based Persian date picker field to the Advanced Custom Fields (ACF) plugin. This allows users to easily select and store dates in the Persian calendar format.

**Features:**

- Persian date selection using a dropdown menu  
- Fully compatible with Advanced Custom Fields  
- Easy to use with no complex setup required  

== Installation ==

1. Download and upload the plugin to the `/wp-content/plugins/` directory.  
2. Go to the WordPress admin panel, navigate to the "Plugins" section, and activate the "ACF Persian Dropdown Calendar" plugin.  
3. When creating or editing ACF fields, select "Persian Dropdown Calendar" as the field type.  

== Frequently Asked Questions ==

= Is this plugin compatible with older versions of ACF? =  

This plugin is designed for ACF version 5 and above. It may work with older versions, but it is recommended to use the latest version of ACF.  

= Can I customize the date field's appearance? =  

Yes, you can customize the appearance of the date field using custom CSS.
